  - 가입한 모임에서는 모임 구성원만의 일정, 가계부, 게시판 등 부가기능으로 모임을 관리할 수 있도록 돕는 앱입니다.

## 👨‍👦‍👦 팀원 소개

- 박성호 (2차 팀장)
- 황운철 (3차 팀장)
- 윤성환 (팀원) : 개인사유로 프로젝트 중단
  <br/>

## 💡 주요 역할

> 담당자 : 박성호

- git 관리
- 메인 페이지
- 모임 관리 (CRUD)
- 모임 신청, 찜하기
- 모임 탈퇴 / 강퇴
- 신청서 / 모임원 관리
- 관리자 페이지
- 랭킹
- 후기
- 404 페이지

> 담당자 : 황운철

- 회원가입
- 로그인
- 정보수정
- 마이페이지
- 나의 후기 관리
- 내모임
  - 게시판

> 담당자 : 윤성환 : 개인사유로 프로젝트 중단

- 내 모임 일정
- 내 모임 게시판
- 내 모임 가계부
